### 4.主要/核心函数分析

#### cmp

```c++
bool cmp(Sum p , Sum q){
    if(p.cnt==q.cnt){
        return p.num<q.num;
    }
    return p.cnt>q.cnt;
}
```

按题目要求输出顺序实现比较函数即可。
